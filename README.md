# 💸 Cash Transactions App

A full-stack web application for managing user accounts and transactions — featuring user authentication, password reset, transaction history, and account balance tracking. Built with **React**, **Node.js**, **Express**, **Sequelize**, **PostgreSQL**, and **Docker**.

---

## 🚀 Features

- 🔐 User Registration & JWT Login
- 💰 Check Account Balance
- 📤 Send & Receive Transactions
- 📄 View Transaction History
- ✉️ Forgot Password & Reset via Email
- 🧪 Seed Data for Demo
- 📘 Swagger API Docs
- 🐳 Fully Dockerized Setup

---

## 📦 Tech Stack

| Layer       | Technology                    |
|-------------|-------------------------------|
| Frontend    | React                         |
| Backend     | Node.js, Express              |
| Database    | PostgreSQL (via Sequelize)    |
| Auth        | JWT, bcrypt                   |
| Email       | Nodemailer                    |
| Validation  | Joi + Zod                     |
| Docs        | Swagger                       |
| Containers  | Docker, Docker Compose        |
